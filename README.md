### üß† Mental Health Screenings

#### Depression Screening (PHQ-9 Style)
- **Clinical-Grade Assessment**: 9-question depression screening tool
- **Severity Levels**: Minimal, Mild, Moderate, Moderately Severe, Severe
- **Crisis Detection**: Special handling for suicidal ideation responses
- **Professional Recommendations**: Tailored advice based on scores

#### Anxiety Assessment (GAD-7 Style)  
- **Validated Tool**: 7-question generalized anxiety disorder screening
- **Comprehensive Scoring**: 0-21 point scale with interpretations
- **Time-Based Assessment**: Focuses on last 2 weeks of symptoms
- **Actionable Results**: Clear recommendations for next steps

#### Stress Evaluation (PSS Style)
- **Perceived Stress Scale**: 8-question stress assessment
- **Coping Assessment**: Includes questions about stress management abilities
- **Balanced Scoring**: Accounts for both stress and coping mechanisms
- **Stress Management Tips**: Built-in guidance for stress relief

## üöÄ Getting Started

### Prerequisites
- Node.js (v14 or higher)
- npm or yarn package manager

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/yourusername/psyche-compass.git
   cd psyche-compass
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start the development server**
   ```bash
   npm start
   ```

4. **Open your browser**
   Navigate to `http://localhost:3000`

### Building for Production

```bash
npm run build
```

This creates an optimized production build in the `build` folder.

## üìã Assessment Details

### Big Five Personality Model
Based on the widely-accepted Five-Factor Model of personality psychology:
- **Extraversion**: Social energy and assertiveness
- **Agreeableness**: Cooperation and trust
- **Conscientiousness**: Organization and dependability  
- **Neuroticism**: Emotional stability and stress reactivity
- **Openness**: Creativity and intellectual curiosity

### Clinical Screening Tools
- **PHQ-9**: Patient Health Questionnaire for depression
- **GAD-7**: Generalized Anxiety Disorder scale
- **PSS**: Perceived Stress Scale for stress assessment

All tools are adapted for educational/screening purposes and include appropriate disclaimers.
